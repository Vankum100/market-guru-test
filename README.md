
## Description
A Simple CRUD Web API with NestJs, Postgres, Sequelize ORM. 

## Installation



## Running the app

- cd into `market-guru-test`
- run `npm install`
- set up your postgres database
- rename `.env.sample` to `.env` and populate the required parameters
- run `npm run start:dev`

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
